#№ О чем проект
____
программа принимает на вход файл формата txt счиатет количество слов в нем и выводит в csv формате сколько раз встерчается каждое слово и его лолю среди всех слов.

#№ Инструкция сборки
____
1) открыть консоль 
2) зайти в папку с проектом
3) ввести "cmake CMakeLists.txt"
4) ввести "cmake --build ."
5) после введенных команд появится папка Debug, в ней будет lab0.exe файл

№№ Инструкция по испольщованию
____
через косоль ввести команду назодясь в папке с exe файлом проекта
команда: leb0.exe <имя вашего txt файла>.txt out.csv

файл с вводными данными должен быть в кадировке ansi
